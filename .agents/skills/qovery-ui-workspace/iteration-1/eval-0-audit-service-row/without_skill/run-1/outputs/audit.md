## Summary

The component leaks raw Tailwind palette classes and a hardcoded hex color, reaches for `ButtonPrimitive` and raw `<button>` instead of the shared `Button`, removes the focus ring, conveys status with color alone, and uses off-scale spacing. The two highest-impact fixes are: (1) replace both action buttons with the shared `Button` component (restoring focus, a11y label, and brand color), and (2) replace the status text with the shared `Status`/`StatusChip` so meaning is conveyed by more than color.
